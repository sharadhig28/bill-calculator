# Bill Calculator
A simple web-based bill calculator for retail shops with GST and discount features.

## Features
- Add multiple items with prices
- Automatic subtotal calculation
- GST computation (configurable %)
- Discount application
- Clean itemized bill display

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Live Demo
[View Project](https://sharadhig28.github.io/bill-calculator/)

## How to Use
1. Enter item name and price
2. Click "Add Item"
3. Apply discount (optional)
4. GST is calculated automatically
5. View final bill with breakdown
